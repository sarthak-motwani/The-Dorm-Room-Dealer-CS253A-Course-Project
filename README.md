```python
pip install django
```

# Installation
```bash
git clone git@github.com:CS253-The-Dorm-Room-Dealer/dormroomdealer.git # ssh
cd dormroomdealer
python manage.py makemigrations
python manage.py migrate
python manage.py runserver
```
